# SJCViewShaker
shaker views AFViewShaker的UIView分类的实现，使用更便捷


## 使用方法

```objective-c

/// shaker for one view

[view sjc_shake];

/// shaker for multiple views

[@[view1, view2, ...] sjc_shake];

```
